# 03. UI 및 사용자 경험 개선 (UI & UX)

**마지막 수정일**: 2026-03-15

본 문서는 `sbe-jira-ui` 프로젝트의 프론트엔드 구성 요소, 인터랙션 디자인 및 사용자 인터페이스 개선 사항을 정의합니다.

---

## 1. 인터페이스 설계 (UI Design)

### 💡 [UI-1] 온보딩 및 접근성 강화
- **배경**: 초기 사용자가 설정 방법을 몰라 이탈하는 문제 해결
- **개선 내용**:
  - 설정 필드 상단에 **발급 가이드 링크** 즉시 노출 (Gemini, Jira)
  - 네트워크 환경 가이드(사내망 vs 테더링) 명시
  - 연결 성공/실패 시 가시적인 배지 및 사용자 이름 표시
- **상태**: ✅ 구현 완료

### 💡 [UI-2] 이슈 카드 및 인라인 액션
- **배경**: 여러 탭을 오가는 불편함(Context Switching) 최소화
- **개선 내용**:
  - **유사 이슈 카드**: 유사도 스코어별 색상 구분 (`score-high`, `score-mid`, `score-low`)
  - **인라인 액션**: 카드 내부에서 즉시 댓글 입력, 상태 변경 드롭다운 노출
  - **Jira 직접 링크**: 이슈 키를 클릭하거나 전용 버튼으로 Jira 본체로 즉시 이동
- **상태**: ✅ 구현 완료

### 💡 [UI-3] AI 에이전트 인터랙션
- **배경**: AI의 동작 과정을 투명하게 보여주어 신뢰도 확보
- **개선 내용**:
  - **Agent Action Card**: AI가 의도를 분석한 결과(JQL 등)를 미리 보여주고 사용자가 '승인 및 실행' 하도록 유도
  - **AI 검증 결과**: 단순 점수 외에 Gemini가 분석한 '유사한 이유'를 텍스트로 함께 출력
  - **초안 편집기**: 자동 생성된 처리 초안을 UI에서 바로 수정하고 등록할 수 있는 전용 텍스트 영역 제공
- **상태**: ✅ 구현 완료

---

## 2. 사용자 경험 기술 (UX Engine)

### 💡 [UX-1] 클라이언트 사이드 데이터 관리
- **배경**: 페이지 새로고침 시 설정 및 대화 내역 휘발 방지
- **구현**: `localStorage` 활용
- **항목**:
  - `gemini_chat_history`: 최대 50턴의 대화 내역 보관
  - `env` (API Keys): 사용자가 입력한 API 키와 사번 보관 (서버 전송용)
  - `similar_users`: 유사 이슈 검색 대상 사용자 목록
- **상태**: ✅ 구현 완료

### 💡 [UX-2] 실시간 진행 피드백
- **배경**: 긴 대기 시간 중 시스템 멈춤으로 오인하는 현상 방지
- **구현**:
  - **Progress Bar**: SSE 데이터를 수신하여 실시간 `%` 및 처리 건수 시각화
  - **Spinners**: AI 응답 대기 시 `dot-spin` 애니메이션 적용
  - **Toasts**: 작업 성공/실패 시 화면 우측 상단에 알림 메시지 출력
- **상태**: ✅ 구현 완료

### 💡 [UX-3] 테마 및 타이포그래피
- **개선 내용**:
  - **Color Tokens**: CSS 변수를 사용한 다크 모드 기반의 고품질 팔레트 적용 (Accent: `#7c83f5`)
  - **Glassmorphism**: 카드 컴포넌트에 반투명 배경 및 Subtle Border 적용
  - **반응형 레이아웃**: 사이드바 기반의 넓은 컨텐츠 영역 확보
- **상태**: ✅ 진행 중 (`5-B` 다크/라이트 모드 전환 로직 미완료)
