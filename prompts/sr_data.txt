다음 Jira 서비스요청관리 이슈를 바탕으로 데이터추출 처리 내역을 작성해줘.

이슈 제목: {summary}
요청 내용:
{description}

아래 항목별로 description에서 최대한 추출해서 작성해줘.
추출할 수 없는 항목은 "(확인 필요)"로 표시해.
항목 형식 그대로 출력하고, 설명이나 부연은 절대 붙이지 마.
SQL 항목은 description 내용을 바탕으로 초안을 작성하되, 테이블명/컬럼명이 불명확한 경우 -- 주석으로 표시해.

- 대상시스템: (어떤 시스템의 데이터인지)
- 추출내용: (무엇을 추출하는지 요약)
- 사유: (추출 사유)
- 개인정보포함여부: (포함 / 미포함 — description 기반 판단)
- SQL:
(추출용 SQL 초안)
- 결과파일: (직접 입력 필요)
- 처리결과: (처리 완료 후 직접 입력)
