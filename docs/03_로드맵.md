## 기술 부채 및 고려사항

| 항목 | 내용 |
|------|------|
| **embedding_cache.json 16MB** | 이슈 수 증가 시 로딩 느려짐. Phase 3에서 SQLite 또는 분할 캐시로 전환 검토 |
| **SSE 구현** | Python stdlib `HTTPServer`는 SSE에 최적화 안됨. `http.server` 대신 `threading` 기반 비동기 처리 필요 |
| **단일 사용자 가정** | 팀 공유 서버로 발전 시 `.env` 분리 및 세션 관리 필요 |
| **Gemini API 비용** | 임베딩 캐시 구축 시 API 호출 비용 발생. 증분 갱신 유지 필수 |
