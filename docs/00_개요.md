## 프로젝트 개요

`sbe-jira-ui`는 사내 Jira와 Google Gemini AI를 연동한 **업무 보조 웹 도구**입니다.  
Python 내장 HTTP 서버 기반으로 로컬에서 실행되며, 브라우저(localhost:8765)로 접근합니다.

### 현재 제공 기능 (v현재)

| # | 기능 | 설명 |
|---|------|------|
| 1 | ⚙ 환경설정 | Gemini API Key, Jira PAT Token 관리 |
| 2 | 🤖 Gemini 채팅 | Gemini와 자유 대화 (히스토리 유지) |
| 3 | 💬 Jira 테스트 | 이슈 키 또는 JQL로 Jira 직접 조회 |
| 4 | 🔍 유사 이슈 검색 | Two-Stage RAG로 미해결 이슈의 유사 완료 이슈 추천 |

### 기술 스택

- **백엔드**: Python 3 (stdlib only, 외부 패키지 없음)
- **프론트엔드**: HTML/CSS/JS (SPA, 단일 파일)
- **AI**: Google Gemini API (채팅, 임베딩, Agent)
- **데이터**: Jira REST API v2, JSON 파일 캐시

---

## 기획 목적

현재 도구는 기술적으로 잘 구현되어 있으나, **실사용자 관점**의 편의성·완성도·확장성에서 개선 여지가 있습니다.  
이 기획서는 실제 업무 환경에서 사용자가 느낄 불편함을 분석하고, 우선순위를 정해 단계적으로 개선하기 위한 로드맵을 제시합니다.
